# 🎧 **Spotify Data Analytics Dashboard (Power BI)**

## 📊 **Project Overview**

This project is an interactive **Spotify Data Analytics Dashboard** built using **Power BI**. It analyzes Top 50 Spotify songs to uncover insights about music trends, popular artists, and track performance.

**Aim :** Turn raw music data into something visual, meaningful, and easy to understand.

---

## 🎯 **Objectives**

- Analyze top-performing songs and artists

- Evaluate artist productivity vs performance (songs vs average rank)

- Identify consistently high-ranking songs (not just popular ones)

- Explore song distribution by album type and explicit content

- Monthly trends in popularity and song releases

- Collaboration insights (artists per song)

Every visual is designed to deliver clear, fast, and decision-oriented insights.

---

## ✨ **Key Features**

- Multi-page interactive dashboard

- Button-based navigation (app-like experience)

- Spotify-themed UI design 🎧

- Combination of KPIs, charts, and tables

- Insight-driven visualizations

## 💡 **Key Insights**

- Some artists produce many songs, but only a few maintain **consistent top rankings**

- **Average rank reveals true performance**, beyond just popularity

- Popularity trends fluctuate across months

- Album type influences song distribution

- Explicit content forms a significant portion of songs

- Some songs involve multiple artists (collaborations)
